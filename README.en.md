# Data Analysis Project

## Project Overview

This project is a data analysis initiative based on the RFM model, utilizing DeepSeek for customer value analysis. RFM analysis is a widely used customer segmentation method that evaluates customers across three dimensions:

- **R (Recency)**: Time of the most recent purchase
- **F (Frequency)**: Purchase frequency
- **M (Monetary)**: Total purchase amount

## Project Files

### RFM_deepseek.ipynb
Jupyter Notebook file containing the core code and implementation logic for RFM analysis.

### render.html
Visualization rendering page displaying the analysis results.

## Technology Stack

- Python
- Jupyter Notebook
- Data analysis libraries (Pandas, NumPy, etc.)
- Data visualization tools

## Usage Instructions

1. Ensure Python 3.x and Jupyter Notebook are installed.
2. Install required dependencies:
   ```bash
   pip install pandas numpy matplotlib
   ```
3. Open the `RFM_deepseek.ipynb` file and execute the analysis code.
4. Open `render.html` in a browser to view the visualization results.

## Analysis Workflow

1. Data loading and preprocessing
2. Calculation of RFM metrics
3. Customer segmentation and scoring
4. Result visualization
5. Business insights and recommendations
